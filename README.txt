Medibit Pharmacy
================

A professional pharmacy management desktop application by Octobit8.

Installation
------------
1. Run the MedibitPharmacySetup.exe installer and follow the on-screen instructions.
2. The application will be installed to your Program Files directory by default.
3. Shortcuts will be created on your Desktop and Start Menu.

First Run & Licensing
---------------------
- On first launch, you will be prompted to enter a license key.
- For a trial, use the key: TRIAL-000000000000 (valid for 7 days).
- For a full license, enter the key provided by Octobit8.

Usage
-----
- Launch the application from the Start Menu or Desktop shortcut.
- Manage inventory, billing, orders, alerts, sales, and settings from the main interface.
- For help, visit the Help or About section in the application.
